# Manufacturing Notes

## PCB

- 4-layer PCB
- L2 solid GND plane
- ENIG recommended if budget allows
- Standard FR-4 acceptable

## Assembly Notes

- Check orientation of all JST connectors.
- Confirm AO4805 pin mapping and orientation before fabrication.
- Confirm DRV8825 exposed pad and thermal vias.
- Confirm polarity markings on dummy battery and power connectors.
- Confirm Teensy module orientation.

## Files to Export

- Gerbers
- Drill files
- BOM
- Pick-and-place file
- Assembly drawing
- Schematic PDF
- DRC report

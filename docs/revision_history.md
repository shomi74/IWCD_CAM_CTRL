# Revision History

| Revision | Date | Description |
|---|---|---|
| Rev A | 2026-05 | Initial IWCD camera control and power board prototype |

## Rev A Main Features

- Teensy 4.0 control
- 8V camera high-side switch
- Bidirectional external battery switch
- Camera switch SSR
- DRV8825 motor driver
- Proximity sensor interface
- I2C environmental/motion sensors
- 4-layer PCB with solid GND plane
